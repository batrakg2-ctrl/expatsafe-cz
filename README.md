# ExpatSafe CZ — Чек-лист иностранца в Чехии

Это интерактивное веб-приложение (MVP), разработанное для помощи иностранцам в Чешской Республике. Приложение позволяет рассчитать список обязательных налогов, страховых взносов и пошлин, а также отслеживать их выполнение в формате удобного чек-листа с сохранением прогресса.

## Архитектура проекта

Проект реализован без сторонних библиотек и тяжелых фреймворков для мгновенной загрузки в любом браузере:
* **[index.html](file:///C:/Users/ideal/.gemini/antigravity/scratch/czech-foreigner-assistant-mvp/index.html)** — интерфейс боковой панели фильтров, дашборда прогресса и контейнера чек-листа.
* **[style.css](file:///C:/Users/ideal/.gemini/antigravity/scratch/czech-foreigner-assistant-mvp/style.css)** — стили оформления (темная тема, эффект стекла glassmorphism, кастомные переключатели, анимации выполнения задач).
* **[app.js](file:///C:/Users/ideal/.gemini/antigravity/scratch/czech-foreigner-assistant-mvp/app.js)** — логика фильтрации задач, расчет сумм и дедлайнов, встроенный интерактивный симулятор Datová Schránka и система автосохранения прогресса.

## Возможности приложения:

1. **Персонализированный фильтр:** Настройте параметры своего пребывания (OSVČ, наём, студент, наличие собаки, автомобиля или недвижимости), и чек-лист мгновенно перестроится под вас.
2. **Интерактивные карточки задач:** Каждая карточка содержит краткое описание, сумму, периодичность, дедлайн уплаты и кнопку «Инструкция ▾», раскрывающую пошаговый гайд с официальными ссылками.
3. **Шкала прогресса:** Наглядно показывает процент выполненных обязательств (например, *«2 из 5 выполнено»*).
4. **Сохранение состояния (LocalStorage):** Все отмеченные вами галочки сохраняются в памяти браузера. Вы можете закрыть страницу, открыть её завтра — и ваш прогресс не потеряется.
5. **Встроенный симулятор Datová Schránka:** Находится прямо внутри карточки соответствующей задачи. Демонстрирует 10-дневный таймер «фикцию доставки» (когда государственное письмо считается официально врученным, даже если вы его не прочитали).
